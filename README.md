# Meme Gallery

Welcome to the Meme Gallery! This is a simple, lightweight, and easy-to-use web application for displaying and interacting with memes.

## Key Features

- Loads a collection of memes from a local directory (as specified in the `MEME_DIR` constant)
- Provides a dynamic grid layout for displaying the memes
- Enables users to open a popup by clicking on a meme for a better viewing experience
- Includes like, share, and report actions for each meme

## Installation

To set up this application, follow these steps:

1. Download the code files provided in this repository or copy the JS code into your project.

2. Replace `'Memes'` in `MEME_DIR = 'Memes';` with the path to your local meme directory containing the images.

3. Ensure you have a container with an id of `memeGrid` in your HTML where the memes will be displayed.

4. Load the application in your preferred web browser.

5. Font Awesome icons are loaded asynchronously from the CDN, so there's no need to install anything extra.

## Usage

After setting up the application, you will see a grid of memes. You can:

- Click on a meme to open a popup for better viewing.
- Click the thumbs-up icon to like a meme.
- Click the share icon to share a meme.
- Click the flag icon to report a meme.

## Dependencies

This project uses Font Awesome for the icons in the meme grid. Icons are loaded asynchronously from the Font Awesome CDN.
